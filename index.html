Get started with tailwind css

  1. Install packages
    2. npm install -D tailwindcss

    This will install tailwind css as a dev dependency

  2. Configure template path
    3. npx tailwindcss init

    It will create your `tailwind.config.js` file ,Add the paths to all of your template files in your `tailwind.config.js` file.

    Your `tailwind.config.js` file should look like this

       /** @type {import('tailwindcss').Config} */
   
  module.exports = {
  
    content: ["./src/**/*.{html,js,jsx,ts,tsx}"],
    
    theme: {
      extend: {},
    },
    
    plugins: [],
  }
  3. Creating vanila css with tailwind css

    4. Now you need a folder to store vanila css generated by tailwind css . create a folder name it css inside assets . Add two files in it `input.css` and `output.css` . You can also add your own css in `input.css` file that will also be added

    your `input.css` file will looks like this 

            @tailwind base;

        @tailwind components;

        @tailwind utilities;

        h1 {
        color: skyblue;
        }
  4. Generate css

    5. open up a terminal and run the code . I wrote the file paths according to my application you can write your own .

    npx tailwindcss -i ./src/assets/css/input.css -o ./src/assets/css/output.css -w

    You can make this step easier by adding the code inside "script" in your `package.json` as a value of "tailwind:watch"

    "scripts" : { "tailwind:watch" : "npx tailwindcss -i ./src/assets/css/input.css -o ./src/assets/css/output.css -w" }

    now npm run tailwind:watch will work

  5. Add tailwind generated css

    6. if you are in a html file link your `output.css` file to html

    Of if you are using React , import `output.css` file in `index.js` file

    import "./assets/css/output.css";
